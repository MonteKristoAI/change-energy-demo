# Change Energy — Demo Site

Next.js 16 + Tailwind v4 + 3-theme switcher (Light / Midnight / Forest) + chatbot widget.

Built by MonteKristo as a redesign demo for [changeenergy.ca](https://www.changeenergy.ca).

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Vercel auto-deploys `main` branch.
